## 🚀 Projetos

<details>
<summary><b>🏪 Sistema de Estoque</b></summary>

Sistema de gerenciamento de estoque via terminal.

**Funcionalidades:**
- Adicionar, listar, buscar e remover produtos
- Busca por ID e por nome
- Cálculo do valor total do estoque
- Valor detalhado por produto
- Validações de entrada

**Conceitos:** Encapsulamento · Getters/Setters · Construtores · Arrays · Scanner

</details>

<details>
<summary><b>📚 Sistema de Biblioteca</b></summary>

Sistema de gerenciamento de biblioteca via terminal.

**Funcionalidades:**
- Adicionar, listar, buscar e remover livros
- Controle de disponibilidade (empréstimo/devolução)
- Aplicação de descontos
- Valor total do acervo

**Conceitos:** Encapsulamento · Sobrecarga · Static · Bloco de inicialização · Arrays · Scanner

</details>
